## 📊 咨询类型

- PHONE - 电话咨询
- ONLINE - 在线咨询
- OFFLINE - 线下咨询
- EMAIL - 邮件咨询

## 📊 咨询状态

- PENDING - 待处理
- PROCESSING - 处理中
- COMPLETED - 已完成
- CONVERTED - 已转案件

## 📝 注意事项

1. AI 评估需要 AI 服务可用
2. 转案件功能需要案件模块支持
3. 所有删除为软删除
