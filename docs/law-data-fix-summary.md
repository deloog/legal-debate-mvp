## 技术细节

### 修复方法

使用关键词匹配规则对法律名称进行分类推断:

```typescript
const KEYWORD_RULES = [
  { pattern: /刑法|刑事诉讼/, category: LawCategory.CRIMINAL },
  { pattern: /劳动合同|劳动法/, category: LawCategory.LABOR },
  { pattern: /专利法|商标法/, category: LawCategory.INTELLECTUAL_PROPERTY },
  { pattern: /公司法|证券法/, category: LawCategory.COMMERCIAL },
  { pattern: /民法典|合同法/, category: LawCategory.CIVIL },
  { pattern: /行政许可|行政处罚/, category: LawCategory.ADMINISTRATIVE },
  { pattern: /环境保护|食品安全/, category: LawCategory.ECONOMIC },
  { pattern: /诉讼法|仲裁法/, category: LawCategory.PROCEDURE },
];
```
