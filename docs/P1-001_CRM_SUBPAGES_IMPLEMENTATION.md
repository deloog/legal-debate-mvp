## 六、已知限制

### 6.1 测试执行限制

由于 Jest 配置 `testPathIgnorePatterns` 忽略了 `[id]` 目录，测试文件在默认配置下无法自动运行。

**解决方案**:

1. 临时修改 `jest.config.js` 移除 `[id]` 忽略规则
2. 或将测试文件移动到 `src/__tests__/app/clients/detail/` 目录
3. 或使用 E2E 测试替代单元测试

### 6.2 表单组件

跟进任务页面使用了占位符表单，因为 `FollowUpTaskForm` 组件尚未实现。

**建议**: 后续迭代中实现 `FollowUpTaskForm` 组件。
