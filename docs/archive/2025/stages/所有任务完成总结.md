## 🚀 如何使用

### 1. 启动应用

```bash
npm run dev
```

### 2. 访问功能

- 合同列表：http://localhost:3000/contracts
- 创建合同：http://localhost:3000/contracts/new
- 合同模板：http://localhost:3000/contract-templates
- 待审批列表：http://localhost:3000/approvals/pending

### 3. 运行测试

```bash
# 运行所有测试
npm test

# 运行组件测试
npm run test:components

# 运行E2E测试
npm run test:e2e
```

### 4. 性能分析

```bash
# 运行性能分析脚本
npx ts-node scripts/performance-analysis.ts
```
