## 🔍 技术细节

### API设计

**端点**: `GET /api/v1/debates/[id]/recommendations`

**查询参数**:
- `limit`: 返回的推荐数量（默认10，最大100）
- `minScore`: 最小推荐分数（0-1之间，默认0）

**响应格式**:
```typescript
{
  success: true,
  recommendations: [
    {
      article: {
        id: string,
        lawName: string,
        articleNumber: string,
        fullText: string,
        category: string,
        // ... 其他字段
      },
      score: number,      // 推荐分数 0-1
      reason: string,     // 推荐原因
      relationType?: string
    }
  ],
  metadata: {
    debateId: string,
    debateTitle: string,
    caseId: string,
    caseTitle: string,
    caseType: string,
    totalCount: number
  }
}
```

### 组件设计

**属性接口**:
```typescript
interface DebateRecommendationsProps {
  debateId: string;           // 必需：辩论ID
  onSelect?: (article: LawArticle) => void;  // 可选：选择回调
  showFilter?: boolean;       // 可选：是否显示过滤器
  limit?: number;             // 可选：推荐数量
  minScore?: number;          // 可选：最小分数
}
```

## 🚀 使用方式

### 在辩论页面中使用

```typescript
import { DebateRecommendations } from '@/components/debate/DebateRecommendations';

// 基本使用
<DebateRecommendations debateId={debateId} />

// 带选择回调
<DebateRecommendations
  debateId={debateId}
  onSelect={(article) => {
    console.log('选择了法条:', article);
  }}
/>

// 带过滤功能
<DebateRecommendations
  debateId={debateId}
  showFilter={true}
/>

// 自定义参数
<DebateRecommendations
  debateId={debateId}
  limit={20}
  minScore={0.8}
/>
```

### API调用示例

```typescript
// 获取推荐法条
const response = await fetch(
  `/api/v1/debates/${debateId}/recommendations?limit=10&minScore=0.5`
);
const data = await response.json();

if (data.success) {
  console.log('推荐法条:', data.recommendations);
  console.log('元数据:', data.metadata);
}
```
