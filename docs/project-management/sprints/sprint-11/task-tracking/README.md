## 📝 使用说明

### 更新任务状态

当任务完成时，请按以下步骤更新文档：

1. 打开对应的阶段追踪文档
2. 找到对应的任务
3. 更新状态：
   - 状态: `⬜ 待开始` → `🟡 进行中` → `✅ 已完成`
   - 测试状态: `⬜ 待测试` → `✅ 已测试`
   - 完成度: `0%` → `100%`
   - 添加完成日期和测试日期
4. 填写测试结果
5. 更新阶段总结表格

### 更新总览文档

当任何阶段的任务状态发生变化时，请：

1. 更新总体进度表格
2. 更新对应阶段的任务列表
3. 更新当前进度百分比
4. 如有必要，调整行动计划
