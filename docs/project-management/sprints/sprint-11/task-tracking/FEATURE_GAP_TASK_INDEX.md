## 📈 质量标准

所有任务必须满足以下质量标准：

- 单元测试覆盖率 > 90%
- 单元测试通过率 = 100%
- 单个文件代码行数 < 500行
- 生产代码禁止使用 `any` 类型
- 测试文件必须放在 `src/__tests__/` 目录

## 📝 使用说明

### 查看详细任务

1. 点击上方的文档链接查看具体分类的详细任务追踪
2. 每个任务包含：子任务进度、文件创建清单、验收标准、测试结果

### 更新任务状态

1. 在具体的任务追踪文档中更新任务状态
2. 标记为"进行中"或"已完成"
3. 更新完成度、实际工时、开始/完成日期
4. 勾选已完成的验收标准
5. 记录测试结果

### 追踪总体进度

- 本文档自动汇总各分类的总体进度
- 查看上方"总体进度"表格了解完成情况
- 定期更新本文档的统计信息
