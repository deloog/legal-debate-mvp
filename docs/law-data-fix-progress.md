## 修复详情

### 1. 修复方法

使用关键词匹配规则对法律名称进行分类推断：

- **刑事类**: 刑法、刑事诉讼、治安管理处罚等
- **劳动类**: 劳动合同、就业促进、工伤保险等
- **知识产权类**: 专利法、商标法、著作权等
- **商法类**: 公司法、证券法、保险法等
- **民法类**: 民法典、合同法、物权法、婚姻法等
- **行政类**: 行政许可、行政处罚、行政复议等
- **经济管理类**: 环境保护、食品安全、药品管理、安全生产等
- **程序法类**: 诉讼法、仲裁法、司法鉴定等

### 2. 修复脚本执行

```bash
npx tsx scripts/fix-law-categories.ts
```

**处理方式**: 批量处理，每批 500 条，逐条匹配关键词规则

**执行时长**: 约 2-3 小时（处理 639,346 条 OTHER 记录）

## 技术细节

### 修复脚本关键代码

```typescript
// 关键词规则匹配
const KEYWORD_RULES = [
  { pattern: /刑法|刑事诉讼|治安管理处罚/, category: LawCategory.CRIMINAL },
  { pattern: /劳动合同|劳动法|就业促进/, category: LawCategory.LABOR },
  {
    pattern: /专利法|商标法|著作权/,
    category: LawCategory.INTELLECTUAL_PROPERTY,
  },
  // ... 更多规则
];

// 批量处理逻辑
while (true) {
  const batch = await prisma.lawArticle.findMany({
    where: { category: LawCategory.OTHER },
    take: 500,
  });

  for (const article of batch) {
    const newCategory = inferCategory(article.lawName);
    if (newCategory) {
      await prisma.lawArticle.update({
        where: { id: article.id },
        data: { category: newCategory },
      });
    }
  }
}
```

- **处理速度**: 约 150-200 条/秒
- **数据库负载**: 中等 (批量更新)
- **内存占用**: 低 (流式处理)
