## 使用指南

### 更新任务状态

在每个任务的追踪文档中，找到对应的任务项，更新以下字段：

1. **状态标记**:
   - ⬜ 待开始 → 🔄 进行中 → ✅ 已完成

2. **测试状态**:
   - ⬜ 未测试 → 🔄 测试中 → ✅ 已测试

3. **完成日期**:
   - 填写实际完成日期（格式：YYYY-MM-DD）

4. **测试日期**:
   - 填写测试执行日期（格式：YYYY-MM-DD）

5. **测试结果记录**:
   - 记录测试结果摘要
   - 记录发现的问题
   - 记录修复情况

### 验收标准检查

每个任务都有明确的验收标准，完成后逐项检查：
- [ ] 标记为 ✅ 表示通过
- [ ] 标记为 ⬜ 表示未通过

### 测试清单执行

每个任务都有详细的测试清单，执行时：
- [ ] 标记为 ✅ 表示测试通过
- [ ] 标记为 ⬜ 表示测试未执行或失败

### 更新阶段总结

每完成一批任务后，更新各阶段文档的"阶段总结"部分：
- 更新完成情况统计表格
- 更新测试覆盖率统计
- 更新质量检查清单
- 记录阻塞问题
- 更新下一步行动
