3. **Mock数据限制**：测试使用模拟数据，实际AI服务未参与，导致提取结果不完整
4. **缓存命中但数据不完整**：虽然缓存命中（显示"缓存命中"日志），但缓存中的数据可能不完整

**日志证据**：
