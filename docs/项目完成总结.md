## 🚀 如何使用

### 1. 应用数据库迁移

```bash
npx prisma migrate dev
```

### 2. 配置环境变量

在 `.env` 文件中添加：

```env
DATABASE_URL="postgresql://..."
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your-email@example.com
SMTP_PASS=your-password
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 3. 启动应用

```bash
npm run dev
```
