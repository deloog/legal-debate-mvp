1. **会员等级数据未初始化**
   - **问题类型**：数据初始化问题
   - **影响范围**：所有依赖会员等级的测试（7个测试）
   - **错误信息**：`data.data.tiers.length = 0`
   - **失败原因**：数据库中没有会员等级数据
   - **解决方案**：
     - 运行会员等级seed脚本：`npm run seed:membership`
     - 或手动执行：`npx tsx prisma/seed-membership.ts`

2. **会员信息API返回错误状态码**
   - **问题类型**：API实现问题
   - **影响范围**：3个测试
   - **错误信息**：`response.status() != 200`
   - **失败原因**：`/api/memberships/me` API实现存在问题
   - **解决方案**：
     - 检查API实现代码
     - 验证数据库连接和查询逻辑
     - 检查中间件配置

3. **使用量统计API返回错误状态码**
   - **问题类型**：API实现问题
   - **影响范围**：2个测试
   - **错误信息**：`response.status() != 200`
   - **失败原因**：`/api/memberships/usage` API实现存在问题
   - **解决方案**：
     - 检查API实现代码
     - 验证使用量记录查询逻辑
     - 检查数据聚合逻辑
